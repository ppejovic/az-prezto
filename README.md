azure-devops
============

[Prezto][1] module for defining aliases and functions for the [Azure CLI][2] [azure-devops][3] extension.



Aliases
-------

### Azure DevOps

  - `azd` is short for `az devops`
  - `azdc` configure
  - `azdl` login
  - `azdL` logout

### Azure Repos

  - `azr` is short for `az repos`

Functions
---------

  - `azr-pr-create` Creates a PR with a title
