az  
=== 

Defines [Azure CLI][1] aliases and functions.   

Aliases     
------- 

### Azure DevOps    

  - `azd` is short for `az devops`

### Azure Repos   

  - `azr` is short for `az repos` 

Functions   
---------   

  - `azr-pr-create` Creates a PR    

[1]: https://docs.microsoft.com/en-us/cli/azure
